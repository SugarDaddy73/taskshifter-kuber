### Setup guide [Windows]

1. Install [Docker Desktop](https://www.docker.com/products/docker-desktop/).

2. Clone this repository:\
**>** `git clone https://USERNAME:TOKEN@github.com/TaskShifter/TaskShifterBackend.git`\
<i>\* Note: replace `USERNAME` with your GitHub username and `TOKEN` with your GitHub personal access token ([classic](https://github.com/settings/tokens)).</i>

3. Build everything up:\
**>** `docker compose up --build`

4. Now you should be able to visit the local Swagger Page:\
**-** http://localhost:7700/swagger/index.html
